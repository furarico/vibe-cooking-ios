# Task Completion Checklist

When a coding task is completed, ensure the following:

1. **Code compiles**: Verify no syntax errors were introduced
2. **Tests pass**: Run `task test` to confirm all tests pass
3. **Style consistency**: Follow the existing Presenter/Screen/Component/Service/Repository pattern
4. **No unused code**: Remove any dead code or unused imports
5. **Backward compatibility**: If editing symbols, check references with `find_referencing_symbols`
